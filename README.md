# 가계부 트렌드 분석기

엑셀 가계부 파일을 업로드하면 월별 지출/수입 트렌드를 시각화하는 React 컴포넌트입니다.

## 기능

- `.xlsx` / `.xls` 파일 드래그 앤 드롭 업로드
- 월별 카테고리별 금액 추이 라인 차트
- 지출 / 수입 / 전체 유형 필터
- 분석 기간 범위 선택
- Y축 최대값 수동 설정
- 카테고리별 합계 · 월평균 · 최대값 요약 카드

## 엑셀 파일 형식

첫 번째 시트의 헤더 행에 아래 열이 필요합니다:

| 열 이름 | 설명 |
|--------|------|
| 날짜 | 엑셀 날짜 시리얼 값 |
| 분류 | 카테고리 (예: 식비, 교통) |
| 금액(원) | 숫자 |
| 수입/지출 | `수입`, `지출`, 또는 `이체지출` |

## 설치 및 실행

```bash
npm install recharts xlsx
```

컴포넌트를 프로젝트에 import하여 사용합니다:

```jsx
import App from './가계부_트렌드';

export default function Page() {
  return <App />;
}
```
